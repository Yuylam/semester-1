# Discrete Structure
This course introduces students to the principles and applications of discrete structure in the field of computer science.

## Course content
- Set theory
- Proof techniques
- Relations
- Functions
- Recurrence relations
- Counting methods
- Graph theory
- Trees
- Finite automata
